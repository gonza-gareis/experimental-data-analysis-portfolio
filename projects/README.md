## Proyectos del portfolio

Esta carpeta contiene una selección de proyectos de análisis de datos experimentales
realizados en laboratorio universitario.

Cada proyecto incluye:
- Descripción del objetivo y del montaje experimental
- Metodología de adquisición y análisis de datos
- Resultados y conclusiones principales
- Informe técnico completo en formato PDF

2. Análisis espectral mediante Transformada de Fourier
