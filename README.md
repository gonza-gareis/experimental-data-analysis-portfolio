# Experimental Data Analysis Portfolio

Este repositorio reúne proyectos de **análisis de datos experimentales** desarrollados en contexto de **laboratorio universitario**, abarcando desde el **montaje y la adquisición de datos** hasta el **análisis, modelado y comunicación de resultados**.

Los proyectos reflejan experiencia en:

- Diseño, **montaje y manejo experimental**
- Adquisición y validación de datos experimentales
- Análisis y limpieza de datos
- Ajuste de modelos y análisis estadístico
- Visualización de datos
- Redacción de informes técnicos y científicos

## Perfil

Formación en **ciencias físicas**, con foco en el **trabajo de laboratorio**, el **análisis de datos** y la **comunicación científica**.

En los proyectos presentados, mi rol principal incluyó:

- **Montaje y ejecución de experimentos**
- Manejo de instrumental y toma de datos
- Análisis e interpretación de resultados
- Redacción de informes técnicos

Con experiencia en trabajo colaborativo en entornos académicos.
