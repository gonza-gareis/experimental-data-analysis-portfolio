# Experimental Data Analysis Portfolio

Este repositorio contiene proyectos de análisis de datos experimentales realizados en contexto de laboratorio universitario.

Los proyectos reflejan experiencia en:
- Análisis y limpieza de datos
- Ajuste de modelos y análisis estadístico
- Visualización de datos
- Redacción de informes técnicos

## Perfil
Formación en ciencias experimentales con foco en análisis de datos y comunicación científica.

Experiencia en trabajo colaborativo, con responsabilidad principal en:
- Análisis de datos
- Interpretación de resultados
- Redacción de informes

- Python
